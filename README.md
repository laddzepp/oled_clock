## Электронные часы

Электронные часы, демонстрирующие московское время. 
Данный проект был разработан с помощью фреймворка ESP-IDF и протокола SNTP.

Компоненты проекта: 
- Контроллер ESP32 CH340C TYPE-C USB
- OLED дисплей (128x64)
